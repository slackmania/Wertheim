# WERTHEIM للتشفير — دليل المستخدم

## 1. مقدمة

**WERTHEIM** هو تطبيق تشفير نصوص يعمل في المتصفح. يمكنك تشفير أي نص بكلمة مرور، وحفظ النص المشفّر في ملف، وإرساله، ويمكن للمستلم فك تشفيره باستخدام كلمة المرور الصحيحة.

**هام:** جميع عمليات التشفير وفك التشفير تتم على جهازك. لا شيء يغادر متصفحك — لا يوجد خادم، ولا رفع بيانات.

---

## 2. التثبيت

**لا حاجة للتثبيت.** فقط افتح ملف HTML واحد:

1. حمّل ملف `wertheim.html`
2. انقر نقرًا مزدوجًا — سيفتح في متصفحك
3. جاهز للاستخدام فورًا

**المتصفحات المدعومة:** Chrome، Brave، Edge، Firefox، Safari، Opera — أي متصفح حديث.

## 3. نظرة عامة على الواجهة

```

│          🔐 WERTHEIM                │
│  تشفير النصوص                        │
├─────────────────────────────────────┤
│  [حقل كلمة المرور]         [👁]     │
├─────────────────────────────────────┤
│  [🔒 تشفير] [🔓 فك تشفير]          │
│  [⇅ تبديل] [📋 نسخ الكل]           │
│  [🗑 جديد / مسح]                    │
├─ لوحة "النص الأصلي" ────────────────┤
│  [📂 فتح] [💾 حفظ]                  │
│  [📋 نسخ] [📋 لصق]                  │
│  [🗑 مسح]                            │
│  ┌────────────────────────────────┐ │
│  │ أدخل النص المراد تشفيره...     │ │
│  └────────────────────────────────┘ │
├─ لوحة "النص المشفّر" ───────────────┤
│  [📂 فتح] [💾 حفظ]                  │
│  [📋 نسخ] [📋 لصق]                  │
│  [🗑 مسح]                            │
│  ┌────────────────────────────────┐ │
│  │ ...يظهر النص المشفّر           │ │
```

### العناصر الرئيسية

| العنصر | الوصف |
|--------|-------|
| **حقل كلمة المرور** | أدخل كلمة المرور هنا. الأزرار غير نشطة بدونها. |
| **👁** | إظهار/إخفاء كلمة المرور |
| **🔒 تشفير** | تشفير النص في اللوحة العلوية، تظهر النتيجة في اللوحة السفلية |
| **🔓 فك تشفير** | فك تشفير النص في اللوحة السفلية، تظهر النتيجة في اللوحة العلوية |
| **⇅ تبديل** | تبديل محتوى اللوحتين |
| **📋 نسخ الكل** | نسخ محتوى كلتا اللوحتين إلى الحافظة (مع العناوين) |
| **🗑 جديد / مسح** | مسح كلتا اللوحتين (بعد التأكيد) |

### شريط أدوات اللوحة (كلتا اللوحتين):

| الزر | الوصف |
|------|-------|
| **📂 فتح** | تحميل ملف من جهاز الكمبيوتر |
| **💾 حفظ** | حفظ محتوى اللوحة إلى ملف (يطلب اسم الملف) |
| **📋 نسخ** | نسخ محتوى اللوحة إلى الحافظة |
| **📋 لصق** | لصق محتوى الحافظة في اللوحة (عند موضع المؤشر) |
| **🗑 مسح** | مسح محتوى اللوحة |

### شريط الحالة:

| العنصر | الوصف |
|--------|-------|
| **INS / OVR** | وضع الإدراج (INS) / وضع الاستبدال (OVR) — التبديل باستخدام مفتاح **Insert** |
| **سطر: / عمود:** | موضع المؤشر الحالي |
| **🧪 اختبارات** | تشغيل الاختبارات التلقائية (تظهر النتائج لمدة 8 ثوانٍ) |

## 4. الاستخدام الأساسي

### تشفير النص

1. أدخل كلمة المرور
2. اكتب أو الصق النص المراد تشفيره في اللوحة العلوية
3. انقر **🔒 تشفير**
4. يظهر النص المشفّر في اللوحة السفلية
5. احفظه (💾) أو انسخه (📋)

### فك تشفير النص

1. أدخل نفس كلمة المرور التي استخدمتها للتشفير
2. ضع النص المشفّر في اللوحة السفلية (فتح، لصق، أو سحب وإفلات)
3. انقر **🔓 فك تشفير**
4. يظهر النص الأصلي في اللوحة العلوية

### إدارة الملفات

**فتح:**
- انقر **📂 فتح** أو اسحب الملف إلى النافذة (سحب وإفلات)
- ملفات `.wtx` تُوضع تلقائيًا في لوحة النص المشفّر؛ إذا كانت كلمة المرور موجودة، يتم فك التشفير تلقائيًا

**حفظ:**
- انقر **💾 حفظ** — يطلب المتصفح اسم الملف
- الحفظ من لوحة النص المشفّر يضيف الامتداد `.wtx` تلقائيًا
- الحفظ من لوحة النص الأصلي يقبل أي اسم/امتداد

---

## 5. الميزات المتقدمة

### السحب والإفلات (Drag & Drop)

اسحب أي ملف نصي إلى النافذة. ملفات `.wtx` تذهب إلى لوحة النص المشفّر، وكل شيء آخر يذهب إلى لوحة النص الأصلي. إذا تم إسقاط ملف `.wtx` وتم تعيين كلمة مرور، يتم فك التشفير تلقائيًا.

### وضع الإدراج / الاستبدال

اضغط على **Insert** للتبديل بين وضع الإدراج والاستبدال. يظهر شريط الحالة الوضع الحالي (`INS` أو `OVR`).

### مفتاح Tab

يقوم مفتاح Tab بإدراج 3 مسافات (بدلاً من نقل التركيز).

### نسخ الكل

يقوم زر **📋 نسخ الكل** بنسخ كلتا اللوحتين إلى الحافظة مع العناوين — مفيد عندما تريد إرسال كل من النص الأصلي والمشفر.

---

## 6. تنسيقات الملفات

| الامتداد | المحتوى |
|-----------|---------|
| `.txt` | أي ملف نصي (بترميز UTF-8) |
| `.wtx` | ملف مشفر بـ WERTHEIM (نص Base64) |

ملفات `.wtx` متوافقة مع تنسيق برنامج Pascal الأصلي.

---

## 7. ملاحظات الأمان

- **لا تنسَ كلمة المرور الخاصة بك** — بدونها، النص المشفّر غير قابل للقراءة بشكل دائم
- تعتمد قوة تشفير WERTHEIM على طول كلمة المرور وتعقيدها
- استخدم كلمات مرور طويلة وعشوائية لأفضل أمان
- التطبيق لا يخزن كلمات المرور ولا يرسل البيانات عبر الشبكة

---

## 8. استكشاف الأخطاء وإصلاحها

| المشكلة | الحل |
|---------|------|
| الأزرار غير نشطة | أدخل كلمة المرور |
| فك التشفير يعطي نصًا مشوشًا | كلمة مرور خاطئة — كلمة المرور غير الصحيحة تنتج مخرجات غير مفهومة |
| الزر حفظ لا يفعل شيئًا | قد يحظر المتصفح التنزيلات — اسمح بها |
| السحب والإفلات لا يعمل | استخدم زر **📂 فتح** كبديل |
| الأحرف مشوشة | تأكد من فتح ملف نصي عادي (وليس `.pdf` أو `.docx`) |
