# WERTHEIM шифровальщик — Руководство пользователя

## 1. Введение

**WERTHEIM** — это браузерное приложение для шифрования текста. С его помощью можно зашифровать любой текст паролем, сохранить зашифрованный текст в файл, отправить его, а получатель, зная пароль, сможет его расшифровать.

**Важно:** Всё шифрование и расшифровка происходят на вашем компьютере. Никакие данные не покидают ваш браузер — нет сервера, нет отправки данных.

---

## 2. Установка

**Установка не требуется.** Достаточно открыть один HTML-файл:

1. Скачайте файл `wertheim.html`
2. Дважды кликните по нему — он откроется в браузере
3. Приложение готово к работе сразу

**Поддерживаемые браузеры:** Chrome, Brave, Edge, Firefox, Safari, Opera — все современные браузеры.

---

├─────────────────────────────────────┤

├─────────────────────────────────────┤
│  [🔒 Зашифровать] [🔓 Расшифровать]  │
│  [⇅ Обменять] [📋 Копировать всё]   │
│  [🗑 Очистить]                      │
├─ Панель "Текст" ─────────────────────┤
│  [📂 Открыть] [💾 Сохранить]         │
│  [📋 Копировать] [📋 Вставить]       │
│  [🗑 Очистить]                      │
│  ┌────────────────────────────────┐ │
│  │ Введите текст для              │ │
│  │ шифрования...                  │ │
│  └────────────────────────────────┘ │
├─ Панель "Зашифрованный" ─────────────┤
│  [📂 Открыть] [💾 Сохранить]         │
│  [📋 Копировать] [📋 Вставить]       │
│  [🗑 Очистить]                      │

### Основные элементы:

| Элемент | Описание |
|---------|----------|
| **Поле пароля** | Сюда вводится пароль. Без него кнопки неактивны. |
| **👁** | Показать/скрыть пароль |
| **🔒 Зашифровать** | Шифрует текст из верхней панели и помещает результат в нижнюю |
| **🔓 Расшифровать** | Расшифровывает текст из нижней панели в верхнюю |
| **⇅ Обменять** | Меняет содержимое панелей местами |
| **📋 Копировать всё** | Копирует содержимое обеих панелей в буфер обмена (с заголовками) |
| **🗑 Очистить** | Очищает обе панели (после подтверждения) |

### Панель инструментов (на обеих панелях):

| Кнопка | Описание |
|--------|----------|
| **📂 Открыть** | Загрузить файл с компьютера |
| **💾 Сохранить** | Сохранить содержимое панели в файл (с указанием имени) |
| **📋 Копировать** | Скопировать содержимое панели в буфер обмена |
| **📋 Вставить** | Вставить содержимое буфера обмена в панель (в позицию курсора) |
| **🗑 Очистить** | Очистить содержимое панели |

### Строка состояния:

| Элемент | Описание |
|---------|----------|
| **INS / OVR** | Режим вставки (INS) / замены (OVR) — переключение клавишей **Insert** |
| **Строка: / Столбец:** | Текущая позиция курсора |
| **🧪 Тесты** | Запуск автоматических тестов (результат отображается 8 секунд) |

## 4. Основы использования

### Шифрование текста

1. Введите пароль в поле пароля
2. В верхнюю панель введите (или вставьте) текст для шифрования
3. Нажмите кнопку **🔒 Зашифровать**
4. Зашифрованный текст появится в нижней панели
5. Его можно сохранить (💾) или скопировать (📋)

### Расшифровка текста

1. Введите пароль (тот же, что использовался при шифровании)
2. Поместите зашифрованный текст в нижнюю панель (открыть, вставить или перетащить)
3. Нажмите кнопку **🔓 Расшифровать**
4. Исходный текст появится в верхней панели

### Работа с файлами

**Открытие:**
- Нажмите **📂 Открыть** или перетащите файл в окно (drag & drop)
- Файлы `.wtx` автоматически помещаются в панель зашифрованного текста и, если задан пароль, сразу расшифровываются

**Сохранение:**
- При нажатии **💾 Сохранить** браузер запросит имя файла
- При сохранении из панели зашифрованного текста расширение `.wtx` добавляется автоматически
- При сохранении из текстовой панели можно указать любое имя и расширение

## 5. Расширенные функции

### Drag & Drop (перетаскивание файлов)

Любой текстовый файл можно перетащить в окно. Файлы `.wtx` попадают в панель зашифрованного текста, все остальные — в текстовую панель. Если перетаскивается `.wtx` файл и пароль уже введён, расшифровка происходит автоматически.

### Режим вставки / замены

Клавиша **Insert** переключает между режимами вставки и замены. Текущий режим отображается в строке состояния (`INS` или `OVR`).

### Клавиша Tab

Клавиша Tab вставляет 3 пробела (фокус не переключается).

### Копирование из обеих панелей

Кнопка **📋 Копировать всё** копирует содержимое обеих панелей в буфер обмена с заголовками — удобно, если нужно отправить и текст, и его зашифрованную версию.

---

## 6. Форматы файлов

| Расширение | Содержание |
|------------|-----------|
| `.txt` | Любой текстовый файл (кодировка UTF-8) |
| `.wtx` | Файл, зашифрованный в WERTHEIM (текст в Base64) |

Файлы `.wtx` совместимы с форматом, используемым оригинальной программой на Pascal.

---

## 7. Замечания по безопасности

- **Не забывайте пароль** — если вы его забудете, зашифрованный текст навсегда останется нечитаемым
- Стойкость шифрования WERTHEIM зависит от длины и сложности пароля
- Рекомендуется использовать длинные случайные пароли
- Приложение не хранит пароли и не передаёт данные по сети

---

## 8. Устранение неполадок

| Проблема | Решение |
|----------|---------|
| Кнопки неактивны | Введите пароль |
| При расшифровке получается не исходный текст | Неверный пароль — неправильный пароль даёт бессмысленный текст |
| При сохранении файла ничего не происходит | Браузер может блокировать загрузку — разрешите её |
| Не работает drag & drop | Используйте кнопку **📂 Открыть** как альтернативу |
| Символы отображаются некорректно | Убедитесь, что вы открываете текстовый файл (не `.pdf`, `.docx`) |
